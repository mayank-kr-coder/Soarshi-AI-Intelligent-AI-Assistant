## 📌 Overview

**Soarshi AI** is a modern AI assistant that combines Google's Gemini AI and Groq LLMs to deliver intelligent conversations, coding assistance, voice interaction, and productivity tools through a clean and responsive web interface.

---

## ✨ Features

- 💬 AI Chat Assistant
- 🎤 Voice Input (Speech Recognition)
- 🔊 AI Text-to-Speech
- 💻 Coding & Programming Assistance
- 🌐 Google Search
- 📅 Date & Time
- 😂 Joke Generator
- 🚀 Open Websites using Voice Commands
- ⚡ Gemini + Groq Integration
- 📱 Responsive UI

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/mayank-kr-coder/Soarshi-AI-Intelligent-AI-Assistant.git

cd Soarshi-AI-Intelligent-AI-Assistant
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Create Environment Variables

Create a `.env` file.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
GROQ_API_KEY=YOUR_GROQ_API_KEY
```

### 4️⃣ Run the Project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
